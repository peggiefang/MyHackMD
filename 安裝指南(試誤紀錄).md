### :traffic_light: Hexo+GitHub 建立網站 (M1 MacBookAir Big Sur 11.3)
> [time=Mon, May 17, 2021 11:21 AM]

**前置：** 
* 下載 Xcode （並開啟、關閉）
* GitHub 建立 repository，user.github.io 。(我的user:peggiefang)

**安裝 node.js** 

1. node.js (https://nodejs.org)

**安裝 Hexo Git**

2. `npm install hexo-deployer-git --save`
3. 確認以下皆安裝
	`node -v`
	`npm -v`
	`Git --version`

**安裝 Hexo**

3. `sudo npm install -g hexo-cli` (安裝 Command line interface for Hexo.)

**初始化 Hexo**

4. 終端 cd 到想存放的目錄下
5. `hexo init blog` (初始化 local 端網站文件夾 blog 名稱可自訂)
6. `cd blog`
7. `sudo npm install`
8. 開啟 剛剛建的 blog 資料夾
9. 開啟文件 `_config.yml`
10. 修改對應代碼，存檔關閉
```
deploy:
  type: git
  repository: https://github.com/peggiefang/peggiefang.github.io.git
  branch: master
```
* **type、repository、branch后均有空格**
11. 終端輸入 `hexo g`  建立靜態檔案，也可以輸入 hexo generate
12. 終端輸入 `hexo d`  部署至 Github Pages，也可以輸入 hexo deploy

* `hexo g` error 則輸入`npm install hexo --save`，並在執行一次`hexo g`
* `hexo d`error 則輸入 `npm install hexo-deployer-git --save`，並在執行一次`hexo d`
13. 到GitHub repository

## Hexo 設定

1. 開啟文件 `_config.yml`
```
title: 我的部落格  #輸入部落格標題
subtitle: 一個抒發本人心情的地方  #輸入副標題
description:  #輸入網站描述 
keywords:  #輸入網站關鍵字(以逗號隔開)，方便 SEO 
author: 王小明  #輸入姓名或暱稱
language: zh-TW  #輸入您所使用的語言
timezone:  #留空以使用系統時間
```

### :traffic_light: python 3.8.5 + PyQt5 + Eric6 (M1 MacBookAir Big Sur 11.3)
> [time=Thu, May 20, 2021 8:23 AM]

https://eric-ide.python-projects.org/eric-download.html

![](https://i.imgur.com/SCrFkhU.png)
1. 點選 Download eric6 進入選擇版本頁面，點了最新的 21.5，下載 eric6-21.5.tar.gz

2. 依序在終端輸入
```
cd ~
python3 -m venv eric_env
~/eric_env/bin/python3 -m pip install --upgrade pip
~/eric_env/bin/python3 -m pip install eric-ide
~/eric_env/bin/eric6_post_install
```

3. 解壓 eric6-21.5.tar.gz
4. 依序在終端輸入 
```
source /Users/chiehfang/eric_env/bin/activate
cd /Users/chiehfang/Downloads/eric6-21.5
sudo python3 install.py
```
安裝完成！
