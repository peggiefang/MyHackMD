# :computer:  錯誤修復紀錄 


## Mac 終端機相關錯誤

### :warning: zsh: command not found: conda


1. 終端輸入 `vim ~/.zshrc`
2. 鍵入 `i` (vim 輸入文字模式)
3. 終端輸入 `source ~/.bash_profile`
4. 鍵入 `Esc` (vim 離開輸入文字模式)
5. `shift` + `:` (vim 輸入指令)
6. 鍵入 `wq` (vim 存檔離開)
7. 終端輸入 `source ~/.zshrc`
8. 終端輸入 `conda --version` (確認是否成功)
